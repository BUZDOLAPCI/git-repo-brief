# Dedalus Marketplace Info

## Short Description
Generate structured briefs for public GitHub repositories

## Long Description
Get repository overviews with stats and topics, extract key files like README and configs, fetch release notes, and view activity snapshots with recent commits and contributors. No auth required. Tools: repo_overview, extract_key_files, release_notes, activity_snapshot.

- Code
- DevOps
